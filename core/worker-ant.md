## v2.0-alpha 阶段1：智能调度

### 核心功能

**一句话指令 → 自动分配给合适的Agent**

```
用户: "帮我设计一个登录页面"
      ↓
任务路由器分析
  • 任务类型: UI设计
  • 匹配关键词: 设计/界面/页面
      ↓
自动匹配: PM(毙杀)
      ↓
执行: hermes chat --profile mibao-kills-pm -Q "设计登录页面"
```

### 使用方式

```bash
# 方式1: 直接输入任务（推荐）
bash scripts/worker-ant-dispatch.sh "帮我设计一个登录页面"

# 方式2: 手动指定Agent（保留）
hermes chat --profile mibao-kills-pm -Q "设计登录页面"
```

### 配置参数

```yaml
# worker-ant-v2.yaml
dispatch:
  auto_mode: true
  fallback_agent: pm

timeout:
  default: 1800
  max: 3600
```

## v1.0 基础功能

### 超时应对

**三级升级机制**

| 级别 | 条件 | 动作 |
|------|------|------|
| L1 | 正常时间内完成 | 正常接收 |
| L2 | L1超时 | 预读关键数据 → 传给新Agent |
| L3 | L2超时 | 诊断日志 → 修复 → 重调度 |

### 配置参数

```yaml
delegation:
  child_timeout_seconds: 1800
  max_iterations: 50
  max_concurrent_children: 3
  max_spawn_depth: 1
```
