# v0.17.0新能力总结

> 基于Hermes v0.17.0官方发布说明，筛选对铁壁四规有实际价值的新能力。

---

## 一、后台异步子代理（工蚁核心改进）

### 功能描述
`delegate_task(background=true)` 现在支持后台异步运行，子代理完成后自动将结果注入新对话轮次。

### 对铁壁四规的价值
- **工蚁v2.0**：多Agent协作从"串行等待"升级为"并行异步"
- **马诺防线**：码仔后台写代码时，米宝可同时处理其他任务
- **混合调度模式**：填补了hermes chat（简单）和kanban（复杂）之间的中间地带

### 使用规范
```python
# ✅ 推荐：独立任务使用异步模式
delegate_task(goal="构建REST API", background=True)

# ❌ 避免：需要用户交互的任务使用异步模式
delegate_task(goal="确认需求", background=True)  # 无法中途交互
```

### 注意事项
- `background=true` 仅用于**独立任务**（不需要中途交互）
- 需要用户确认的任务仍用同步模式
- 子代理完成后自动将结果注入新对话轮次

## 二、内存批量原子操作（历史书核心改进）

### 功能描述
`memory` 工具新增 `operations` 数组，支持一次调用中同时执行add/replace/remove操作，原子执行，不会中途失败。

### 对铁壁四规的价值
- **历史书**：L1 MEMORY.md压缩流程从"多次脆弱调用"变为"一次可靠操作"
- **解决痛点**：之前压缩时偶尔中途失败，现在彻底解决

### 使用示例
```python
# ✅ 推荐：单次原子操作
memory(action="add", target="memory", operations=[
    {"action": "remove", "old_text": "过时条目1"},
    {"action": "remove", "old_text": "过时条目2"},
    {"action": "add", "content": "新条目"}
])

# ❌ 传统方式：多次调用（可能中途失败）
memory(action="remove", target="memory", old_text="过时条目1")
memory(action="remove", target="memory", old_text="过时条目2")
memory(action="add", target="memory", content="新条目")
```

## 三、文件系统回滚保护（图书馆安全改进）

### 功能描述
`checkpoints.enabled: true` 启用文件系统快照，代码操作前自动创建快照，使用 `/rollback [N]` 可回滚到之前的状态。

### 对铁壁四规的价值
- **图书馆**：项目文件操作更安全，出错可回滚
- **马诺防线**：代码审查过程中的文件安全保护

### 配置状态
```yaml
checkpoints:
  enabled: true  # 已启用
  max_snapshots: 20
  max_total_size_mb: 500
  retention_days: 7
```

### 使用方式
- 自动创建：代码操作前自动创建快照
- 手动回滚：`/rollback [N]` 回滚到指定快照
- 查看快照：`/snapshot list` 查看可用快照

## 四、read_file新能力（马诺防线增强）

### 功能描述
`read_file` 现在支持提取以下格式为可读文本：
- `.ipynb`（Jupyter笔记本）
- `.docx`（Word文档）
- `.xlsx`（Excel表格）

### 对铁壁四规的价值
- **马诺防线**：审查代码时可直接读取项目文档，无需额外工具转换
- **图书馆**：文档管理更方便，支持多种格式

### 使用示例
```python
# ✅ 直接读取Jupyter笔记本
read_file(path="analysis.ipynb")

# ✅ 直接读取Word文档
read_file(path="requirements.docx")

# ✅ 直接读取Excel表格
read_file(path="data.xlsx")
```

## 五、视频生成（新工具）

### 功能描述
`video_generate` 工具支持文生视频、图生视频、多图视频生成和关键帧动画。

### 对铁壁四规的价值
- **游戏项目**：可用于生成游戏角色动画、场景过渡效果、宣传视频
- **内容创作**：抖音视频素材生成

### 配置状态
```yaml
video_gen:
  provider: agnes  # 已配置Agnes Video V2.0
```

### Agnes Video V2.0特性
- 免费期间：$0/秒
- 支持：文生视频、图生视频、多图视频、关键帧动画
- 最长视频：18秒（441帧/24fps）

## 六、Curator零成本常规运行

### 功能描述
技能整理的常规扫描不再消耗辅助模型token，只有显式调用 `--consolidate` 才用LLM。

### 对铁壁四规的价值
- **历史书**：60个技能的维护零token消耗
- **成本优化**：后台技能维护不再消耗辅助模型预算

### 配置状态
```yaml
curator:
  enabled: true
  consolidate: false  # 常规运行零token
  prune_builtins: true
```
